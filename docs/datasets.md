# Documentation métier

La documentation métier canonique des familles de données publiques est maintenue sur `1dex.fr`:

<https://1dex.fr/developpeurs/api#donnees>

Ce dépôt connecteur documente uniquement l'usage des clients. Le sens des sources, les règles d'exposition publique et les limites doivent être mis à jour dans la documentation runtime d'Explorer.
